Welcome to Citation Needed

Citation needed is a simple firefox extension that records your traversal of Wikipedia. Currently the graph is displayed in the browser. There are future plans to allow for exporting it, in a variety of formats.

History/Motivation

The Citation Needed extension was developed for Ryan and Johnny's Podcast which is a very serious podcast that covers topics read on Wikipedia.

Installation and Use

Just click the button on the left to install the extension. Mozilla browsers only! To use the extension, simply open the Citation Needed view from the 'Tools' menu. Browse Wikipedia, and a graph will be rendered there. You can even tear the graph tab out and view it in a separate window if you like.

Developers

Building the extension requires Python and the Mozilla Add-on SDK. Citation Needed requires several SDK extensions, so if you've already got Python installed, your best bet is to use the get_sdk.py script included in the source:

python get_sdk.py

Will fetch a current Mozilla SDK, all the extensions required to build Citation Needed, and install them in a folder called sdk. To get started with the extension cd to the sdk/addon-sdk-1.x directory, and 

source bin/activate

or if you're on Windows run:

bin/activate.bat

You're all set! To run a test installation in Firefox, simply use the cfx tool:

cfx run

To build the extension, use GNU make:

make xpi

This not only builds the extension, but produces the SHA1 hashfile needed for installation from this site. Don't commit the extension without committing the hashfile that goes with it!
